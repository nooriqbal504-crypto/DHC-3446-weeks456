## 🔵 Week 4 — Advanced Threat Detection

### Rate Limiting (`routes/rateLimiter.ts`)
```ts
// Blocks brute-force — 5 login attempts per IP per 15 minutes
export const loginLimiter = rateLimit({ windowMs: 15*60*1000, max: 5 })
```

### API Key Auth (`middleware/apiKeyAuth.ts`)
```ts
// Every sensitive API call must include: X-API-Key: <valid-key>
const apiKey = req.headers['x-api-key']
if (!VALID_API_KEYS.has(apiKey)) res.status(403).json({ error: 'Invalid API key.' })
```

### CORS Restriction (`routes/rateLimiter.ts`)
```ts
// Only requests from our trusted frontend are accepted
origin: process.env.ALLOWED_ORIGIN ?? 'http://localhost:3000'
```
